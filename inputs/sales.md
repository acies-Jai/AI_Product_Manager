# Growth & User Acquisition Team — Context

## Team
Growth Head (Rithvik Nair) + 8 growth managers + 2 analysts. Own user acquisition, activation, retention, and monetisation levers.

## Current Funnel Metrics (monthly)
- App installs: 4.2M
- Install-to-first-order conversion: 31% (industry benchmark: 42%)
- D7 retention (users who place 2nd order within 7 days): 44%
- D30 retention: 28%
- Avg orders per active user per month: 6.1

## Key Acquisition Channels
- Performance marketing (Meta/Google): 62% of installs, ₹180 CAC
- Referral programme: 22% of installs, ₹45 CAC
- Organic/SEO: 11% of installs
- OTT / brand: 5%

## Growth Pain Points
1. **High install-to-order drop-off**: New users abandon during checkout — onboarding flow is confusing for first-timers
2. **Referral programme underperforming**: Referral redemption rate only 38%; deep link breaks on some Android versions
3. **Search failure rate**: 21% of searches return 0 results — users churn after failed search
4. **No win-back flow**: Lapsed users (no order in 30 days) get generic push notifications, not personalised

## Opportunities Flagged by Growth Team
- Personalised home screen (currently static) — estimated +8% order rate
- Smart search with regional language + typo tolerance — estimated -15% search abandonment
- Streamlined first-order flow (reduce steps from 5 to 2)
- Reactivation nudges based on purchase history

## Competitive Observations
- Blinkit has 1-click reorder feature driving 30%+ of its orders
- Swiggy uses dietary preference filters to increase cart value by ~₹80/order
